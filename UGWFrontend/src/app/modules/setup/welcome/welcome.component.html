Ultimate Gym Wue

DEMO MODE

As BGW's main intent was to provide a code base that theoretically could be picked up by anyone that knows how to spin up a node server UGW's intent was to create an ultimate platform. My plans included notifications for changes on the substitution plans, a collective calendar and so on. The backend was written in rust and node and also did the analysis needed. This enabled heavy caching which brought response times down to 5ms on the same machine. This project died because of the same reason BGW did. For the first time I took testing seriously and came up with a 95.5% coverage. In hindsight rocket.rs was the wrong framework for rust, Actix-Web is way better. This demo version of the app mocks every backend connection.
